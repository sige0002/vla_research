## 2. Physical Intelligenceの主要技術

### 2.1 MEM（Multi-Scale Embodied Memory）

最大15分のタスクに対応する二重経路メモリシステム。

**短期メモリ（ビデオエンコーダ）**:
- ViTに空間-時間分離アテンション追加（4層ごとに因果的時間アテンション）
- 追加パラメータなし、正弦波時間位置エンコーディング
- 18フレーム（54秒）をリアルタイム処理可能
- 計算量: O(Kn^2 + nK^2)

**長期メモリ（言語ベース）**:
- 完了イベントの自然言語要約 → VLMが圧縮・更新
- **成功時のみ更新**（ナイーブ連結は20-25%性能低下）
- レシピ準備タスクで2-3倍改善、物体探索95%

### 2.2 RLT（RL Tokens）

VLA内部表現から情報ボトルネックとしてRLトークンを抽出し、精密操作を高速改善。
- アクションを「編集」する設計（置換ではない）
- 実機上15分〜2時間で学習完了
- M3ネジ締め8.2倍、ケーブル挿入4.4倍の改善
- 人間テレオペ速度を超える場合も

### 2.3 RECAP（RL with Experience and Corrections）

アドバンテージ条件付けによるVLAのRL改善。
- テキスト入力に「Advantage: positive/negative」を追加
- 分類器フリーガイダンスで推論時にガイド
- 価値関数: 201ビン離散化分布的価値関数
- エスプレッソ製造でスループット2倍以上

## 4. 研究テーマ候補（PI以外の技術を活用）

### テーマA: 離散拡散 + テスト時再帰精製

DIVA/DD-VLAの並列アクション生成 + RD-VLAの再帰的精製。難しいタスクで反復回数を増やし、簡単なタスクは1パスで完了。PIのFlow Matchingとは根本的に異なるアプローチ。

### テーマB: ワールドモデル共進化型報酬生成

VLAWの共進化 + LRMのVLM報酬生成。Qwen3.5をポリシー・ワールドモデル・報酬モデルの三位一体で活用。自律的改善ループの構築。

### テーマC: Embodied Multimodal CoT + 3D空間推論

HALOのEM-CoT + SpatialVLA/Any3D-VLAの3D表現。テキスト推論→3D空間サブゴール→アクションの階層的推論チェーン。

### テーマD: MEM + RLT + RECAPの統合（元の構想ベース）

PIが個別発表した技術の統合。Qwen3.5-4B + MEM短期/長期メモリ + RLT精密制御 + RECAPアドバンテージ条件付け。
